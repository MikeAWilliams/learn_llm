## 01_bigram_model

In this video titled "The spelled-out intro to language modeling: building makemore" he build up a bigram model, which is a model that
considers only two letters at a time. It predicts the second letter given the first. He is using names.txt as input which can be
download from https://github.com/karpathy/makemore/blob/master/names.txt

The video can be found https://www.youtube.com/watch?v=PaCmpygFfXo
